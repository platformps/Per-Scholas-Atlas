# `apps/web/public/`

Static assets served at the root of the deployed site.

## Required files

- **`per-scholas-logo.png`** — Per Scholas 30th Anniversary horizontal logo
  with the geometric "30" mark + stacked "PER SCHOLAS" wordmark.
  Used in the headers of `/login`, `/`, and `/admin`.
  Source: Per Scholas brand asset library (May 2025 brand book).

## Optional

- **`favicon.ico`** — site favicon. Currently using Next.js default.
- **`atlas-mark.svg`** — Atlas-specific product mark, if/when designed.
